# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Explore fundamental Python collections—lists, dictionaries, sets, and tuples—by using them to store, access, and manipulate data. This assignment helps students learn when to choose each type and how to perform common operations.

## 📝 Tasks

### 🛠️ Working with Lists

#### Description
Create a list of at least five items (e.g., student names, colors, or numbers) and write functions to add, remove, and search for elements.

#### Requirements
Completed program should:

- Initialize a list with sample values
- Implement functions to append, insert, and delete elements
- Search the list for a given value and report its index
- Iterate over the list and display each item

### 🛠️ Using Dictionaries

#### Description
Build a dictionary representing a simple inventory or phonebook. Practice accessing, adding, and updating key-value pairs.

#### Requirements
Completed program should:

- Define a dictionary with at least three key-value pairs
- Add a new key-value pair based on user input
- Update the value of an existing key
- Safely handle lookups with error checking

### 🛠️ Sets and Tuples in Action

#### Description
Demonstrate the use of sets for unique collections and tuples for fixed data. Compare operations on both types.

#### Requirements
Completed program should:

- Create a set from a list with duplicate entries and show that duplicates are removed
- Perform union, intersection, and difference operations on two sets
- Define a tuple containing related values and attempt (and explain) why it cannot be modified
- Convert a tuple to a list, modify it, then convert back to a tuple
